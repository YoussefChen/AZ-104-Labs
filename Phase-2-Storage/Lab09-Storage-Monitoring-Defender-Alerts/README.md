# Lab 09 - Storage Monitoring, Microsoft Defender for Storage & Alert Configuration

This lab focuses on enhancing the security posture of Azure Storage by enabling Microsoft Defender for Storage, configuring diagnostics and log collection, and setting up custom alerting rules.

## Objectives
- Enable threat detection on storage resources
- Stream diagnostic logs to a Log Analytics workspace
- Create alerts to notify on suspicious storage activity

## Azure Services Used
- Azure Storage Account
- Microsoft Defender for Cloud
- Log Analytics Workspace
- Azure Monitor Alerts
